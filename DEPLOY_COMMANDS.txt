## If Something Fails

### "adb: command not found"
First, download ADB:
```powershell
# Install Android SDK Platform Tools
# Download from: https://developer.android.com/studio/releases/platform-tools
# Add to PATH, or use full path: C:\adb\adb.exe
```

### "Cannot connect to device"
```powershell
# Check device:
adb devices

# Your phone should show here. If not:
# 1. Enable USB Debugging on phone
# 2. Check Settings > Developer Options
# 3. Tap "Revoke USB Debugging Authorizations"
# 4. Reconnect and authorize again
```

### App won't install
```powershell
# Remove old version first:
adb uninstall com.etrsystems.axisight

# Then install:
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### App crashes on launch
```powershell
# Check errors:
adb logcat > error.txt

# Reinstall fresh:
adb uninstall com.etrsystems.axisight
adb install app/build/outputs/apk/debug/app-debug.apk
```
